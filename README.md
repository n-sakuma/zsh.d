zsh.d
=====


# dependence (coreutils)

## for homebrew

```
brew install coreutils
```

## for MacPorts

```
sudo port install coreutils
```

# Set up

```
git clone git://github.com/n-sakuma/zsh.d.git ~/.zsh.d

ln -s ~/.zsh.d/dot.zprofile ~/.zprofile

cp ~/.zsh.d/env/switch_view.zsh{.sample,}
```

Enjoy!
